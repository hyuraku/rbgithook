# Rbgithook

Simple Git hook made by Ruby

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rbgithook'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install rbgithook

## Usage

```
rbgithook install
rbgithook add <file> <cmd> ex. rbgithook add pre-commit "rspec spec/"
rbgithook uninstall
```
